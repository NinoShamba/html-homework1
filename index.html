Главные языки фронтенда

HTML

HTML(Hyper Text Markup Language) - это язык разметки, который отвечает за контент на наших веб-страницах. HTML включает в себя множество различных элементов: текст, кнопки, таблицы, списки и пр.

CSS

CSS (Cascading Style Sheets) - каскадные таблицы стилей; язык, который отвечает за внешний вид наших веб-страниц. С помощью CSS мы можем управлять множеством вещей: цвета, градиенты, шрифты, размеры элементов и пр.

JavaScript

JavaScript - язык программирования, который широко применяется не только во фронтенде, но и в бэкенде, и этим не ограничивается. Придает нашим веб-страницам интерактивности.
